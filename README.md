# WeatherToGo

WeatherToGo は、日々の外出計画において「天気に合わせた行動判断を簡単にしたい」という課題を解決するアプリ

---

## ✨ 主な機能

### 1️⃣ 位置検索（ジオコーディング）

- 市区町村名を入力して地点を検索
- Open-Meteo Geocoding API を利用して緯度・経度を取得

### 2️⃣ 天気表示（今日／明日）

- 気温、体感温度、降水確率、風速、視程、降雪量などをカード形式で表示

### 3️⃣ おでかけ提案ロジック

- 天候データをもとに行動プランを自動提案
- 例：
  - 降水確率 70％以上 → 屋内推奨
  - 風速 10m/s 以上 24m/s 以下 → 控えめ外出
  - 晴天で気温 15〜28℃ → 屋外推奨

### 4️⃣ お気に入り地点（今後実装予定）

- お気に入り登録／解除機能
- `localStorage`で保持し、一覧からワンクリックで再表示

### 5️⃣ 表示・操作性

- Tailwind CSS による柔軟なレスポンシブ対応
- 滑らかな横スクロールとアニメーション

| **CheckWeather.tsx** | 取得した天気データの詳細表示と行動プラン提示             |

---

## 🧮 主要ロジック

### 天気データ取得

- `getWeather()`関数で Open-Meteo API から天気データを取得
- 平均風速算出やデータ整形後、`WeatherInfomation`型に格納

### 条件判定

- `judgementWeather()`：気温・降水確率・風速・降雪量などから総合的に天気を判定
- `setWeatherIcon()`：条件に応じて SVG アイコンを設定
- `suggestPlan()`：その日の天気に応じた行動プラン（屋内／屋外など）を生成
